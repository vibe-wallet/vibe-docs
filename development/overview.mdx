## Monorepo Structure

Vibe Wallet is organized as a monorepo with four packages:

| Package | Description | Stack |
|---------|-------------|-------|
| `extension/` | Chrome extension | Vite + React + CRXJS |
| `server/` | MCP server | Express + WebSocket + SSE |
| `dashboard/` | Monitoring UI | Vite + React |
| `web/` | Landing page | Next.js 16 |

Each package is independently buildable and has its own `package.json`.

## Prerequisites

- **Node.js** 18 or later
- **pnpm** package manager
- **Chrome** or **Chromium** browser (for extension development and testing)

## Quick Start

Clone the repository and install dependencies in each package:

```bash
git clone https://github.com/anthropics/vibe-wallet.git
cd vibe-wallet

# Install dependencies for each package
cd extension && pnpm install && cd ..
cd server && pnpm install && cd ..
cd dashboard && pnpm install && cd ..
cd web && pnpm install && cd ..
```
