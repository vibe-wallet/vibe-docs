# Vibe Wallet MCP Tools

Vibe Wallet exposes 27 MCP tools organized into six categories. These tools allow AI agents to manage wallets, send transactions, query balances, and interact with smart contracts across EVM and Solana networks.

## Approval Required

The following tools modify on-chain state and require explicit user approval before execution:

- `wallet_sendTransaction` -- Send ETH
- `wallet_signMessage` -- Sign an EVM message
- `wallet_sendToken` -- Send ERC-20 tokens
- `wallet_approveToken` -- Approve ERC-20 token spending
- `solana_sendTransaction` -- Send SOL
- `solana_signMessage` -- Sign a Solana message

All other tools are read-only or manage local wallet state and do not require approval.
