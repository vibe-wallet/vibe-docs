# wallet_getBlockNumber

Get the latest block number on the currently connected EVM network.

## Parameters

This tool takes no parameters.

## Returns

A hex-encoded string representing the latest block number (e.g., `"0x10D4F1E"`).

## Example

```
"What's the latest block number?"
```

```
"Get the current block height"
```

## Notes

- No approval required.
- The result is hex-encoded. Convert to decimal for a human-readable block number.
- This is an EVM-only tool.
