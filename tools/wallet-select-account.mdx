# wallet_selectAccount

Switch the active account by name. Subsequent operations will use the selected account.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `accountName` | string | Yes | The name of the account to activate |

## Returns

A confirmation string indicating the newly active account.

## Example

```
"Switch to my account called 'trading'"
```

```
"Select the 'main' account"
```

## Notes

- Use `wallet_listAccounts` first to see available account names.
- No approval required.
- Works with both EVM and Solana accounts.
