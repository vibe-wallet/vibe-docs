# wallet_selectInstance

Select a specific wallet instance to control. All subsequent wallet operations will target the selected instance.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `instanceId` | string | Yes | The unique ID of the wallet instance to select |

## Returns

A confirmation string indicating the selected instance.

## Example

```
"Switch to wallet instance abc123"
```

```
"Select the second wallet"
```

## Notes

- Use `wallet_listInstances` first to get available instance IDs.
- No approval required.
- Works with both EVM and Solana account types.
